# 🤖 Mirai Edge Workspace

欢迎来到 Mirai 边缘端工作区！
这里是你的物理设备（如树莓派）或虚拟环境（如游戏引擎）与大模型大脑沟通的桥梁。

## 📁 目录结构
* `.env`: 环境变量配置。你可以在这里修改本设备的名称和大脑的连接地址。
* `mirai_tools/`: **核心目录**。你所有的自定义工具函数都应该放在这里面。

## 🛠️ 如何添加新能力（让大脑学会新技能）？

1. 在 `mirai_tools/` 目录下新建一个 `.py` 文件（或者直接修改 `example_tool.py`）。
2. 编写你的 Python 函数。
3. **【最重要的一步】**：务必为你的函数写好 **Docstring（文档注释）** 并在参数上标明**类型提示**。大脑完全依赖这些注释来理解何时、如何调用你的工具。
